# Emergent Embodied AI

Language: 中文 | [English](README.en.md)

一个面向学习、研究和实践的具身智能知识库，按“一个机器人智能系统从输入到落地的技术环节”组织。

本仓库不只是收集链接，而是帮助读者建立一张清晰地图：具身智能系统由哪些部分组成，每个部分解决什么问题，常见方法是什么，前沿论文、系统案例和开源项目分别落在哪一层。

## 学习主线

建议按下面顺序阅读：

1. [00-start-here](00-start-here/README.md)：先知道具身智能是什么，以及这个仓库怎么用。
2. [01-system-overview](01-system-overview/README.md)：看懂一个具身智能系统的全链路。
3. [02-perception](02-perception/README.md)：机器人如何感知环境。
4. [03-representation](03-representation/README.md)：如何表示空间、物体、状态和动作。
5. [04-data](04-data/README.md)：机器人学习数据从哪里来，为什么贵。
6. [05-policy-learning](05-policy-learning/README.md)：机器人策略如何从数据和交互中学出来。
7. [06-planning-and-control](06-planning-and-control/README.md)：如何把目标变成可执行动作。
8. [07-simulation-and-evaluation](07-simulation-and-evaluation/README.md)：如何用仿真、benchmark 和实机测试能力。
9. [08-hardware-and-deployment](08-hardware-and-deployment/README.md)：硬件、部署、安全和运维问题。
10. [09-frontier-tracking](09-frontier-tracking/README.md)：论文、项目、机构、数据集和开源生态跟踪。
11. [10-case-studies](10-case-studies/README.md)：代表性系统和项目拆解。
12. [11-glossary](11-glossary/README.md)：术语表。
13. [12-roadmaps](12-roadmaps/README.md)：学习路线、研究路线和技术路线。
14. [99-templates-and-notes](99-templates-and-notes/README.md)：论文笔记、主题综述和问题卡片模板。

## 技术环节

```text
任务目标
→ 感知环境
→ 表示世界和动作
→ 收集与组织数据
→ 学习策略
→ 规划与控制
→ 仿真和评测
→ 硬件部署
→ 前沿跟踪和案例复盘
```

## 怎么使用

1. 如果你刚进入这个方向，先读 [00-start-here/overview-map.md](00-start-here/overview-map.md)。
2. 顺着“学习主线”快速浏览每个目录的 README。
3. 遇到术语先查 [11-glossary](11-glossary/README.md)。
4. 想看前沿进展，再进入 [09-frontier-tracking](09-frontier-tracking/README.md)。
5. 想看完整系统，阅读 [10-case-studies](10-case-studies/README.md)。
