# 99 Templates And Notes

这个目录用于放论文笔记、主题综述、问题卡片和临时洞察。

## 模板

- [templates/paper-note.md](templates/paper-note.md)：论文笔记模板。
- [templates/topic-brief.md](templates/topic-brief.md)：主题综述模板。
- [templates/question-card.md](templates/question-card.md)：问题卡片模板。

## 建议

- 精读笔记写在这里。
- 主题成熟后，把稳定结论迁移到对应技术环节目录，例如 `03-representation`、`05-policy-learning`、`07-simulation-and-evaluation`。
- 不确定的想法可以先留在问题卡片里。
