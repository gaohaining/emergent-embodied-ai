# Evaluation Basics

## 评测目标

具身智能评测不只看单次任务成功，还要看泛化、鲁棒性、安全性和成本。

## 评测维度

- 成功率：任务是否完成。
- 泛化性：新物体、新场景、新指令、新干扰下是否有效。
- 鲁棒性：遮挡、噪声、碰撞、偏差、失败恢复。
- 效率：时间、动作数量、能耗、计算资源。
- 安全性：是否避免损坏环境、伤害人或进入不可恢复状态。
- 可复现性：硬件、环境、数据和代码是否可复现。

## 评测风险

- benchmark 被训练集污染。
- demo 只展示成功样本。
- 任务定义过窄，无法代表真实使用。
- 不同机器人平台之间结果不可比。
