# Data

## 数据来源

- 人工遥操作数据。
- 机器人自主探索数据。
- 仿真数据和合成数据。
- 人类视频和互联网视频。
- 失败案例和纠错数据。
- 多机器人、多环境、多任务数据集。

## 质量维度

- 动作是否可执行。
- 观察和动作是否时间同步。
- 是否包含失败和恢复。
- 是否有任务、语言、对象、场景标注。
- 是否覆盖长尾环境。
- 是否能跨硬件复用。
