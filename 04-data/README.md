# 04 Data

数据解决的问题是：机器人从哪里获得经验，以及这些经验如何被模型学习。

## 初学者先读

- [data.md](data.md)：数据来源、质量和规模化。

## 数据来源

- 人类遥操作机器人。
- 机器人自主采集。
- 仿真生成。
- 人类视频和互联网视频。
- 失败日志和人工纠错。

## 为什么机器人数据难

- 真实机器人采集慢、贵、容易损耗硬件。
- 不同机器人动作空间不同，数据不容易合并。
- 失败数据很重要，但很难系统记录。
- 视频里有人类动作，不等于机器人能直接执行。

## 初学者要问

- 数据来自真实机器人还是仿真？
- 有动作标签吗？
- 动作和图像是否时间对齐？
- 数据能跨机器人复用吗？
- 数据许可是否允许训练模型？
