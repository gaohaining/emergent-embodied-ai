# State And Scene

## 状态是什么

状态是系统认为“完成任务需要知道的信息”。它不一定等于真实世界的全部信息。

## 常见状态

- 机器人本体状态：关节角、速度、末端位姿。
- 物体状态：位置、姿态、类别、是否被抓住。
- 场景状态：地图、障碍物、可通行区域。
- 任务状态：当前子任务、进度、失败原因。

## 场景表示

- 图像或视频。
- 点云或 3D mesh。
- 对象列表和空间关系。
- 语义地图。
- latent representation。
