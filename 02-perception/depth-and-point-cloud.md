# Depth And Point Cloud

## 它解决什么

深度和点云帮助机器人理解三维空间结构，例如物体距离、形状、表面和障碍物。

## 常见输入

- RGB-D 相机。
- LiDAR。
- 多视角重建。
- 仿真环境中的精确几何信息。

## 为什么重要

- 抓取和避障需要空间位置。
- 移动机器人需要地图和可通行区域。
- 机械臂需要知道目标相对末端执行器的位置。

## 常见难点

- 透明、反光、柔性物体难感知。
- 遮挡会导致点云不完整。
- 传感器噪声会影响控制。
