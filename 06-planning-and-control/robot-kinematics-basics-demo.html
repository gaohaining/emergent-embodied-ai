机器人运动学基础演示：正解 FK 与逆解 IK

这个小演示用二维两连杆机械臂解释基础直觉：正解是“关节角度 -> 末端位置”，逆解是“目标位置 -> 关节角度”。 它不是工业级求解器，只用来帮助理解正文里的概念。

机械臂视图

当前机械臂 IK 目标 另一个可能逆解

1. 正解：调关节，看末端

滑动肩关节和肘关节，或直接拖动画布里的肘关节/末端控制点。你给的是关节空间里的角度，系统算出末端在平面中的 x/y 位置。

正解 FK 公式

x = L1 cos(θ1) + L2 cos(θ1 + θ2) y = L1 sin(θ1) + L2 sin(θ1 + θ2)

2. 逆解：设目标，看关节

可以拖动画布里的橙色目标点，也可以用下面的滑杆精确设置。目标会被限制在可达圆环内。

逆解 IK 公式

d² = x² + y² cos(θ2) = (d² - L1² - L2²) / (2 L1 L2) θ1 = atan2(y, x) - atan2(L2 sin(θ2), L1 + L2 cos(θ2))
  1. 拖绿色肘关节点：主要改变肩关节角度。
  2. 拖绿色末端点：改变肘关节角度，并保持第二段连杆长度。
  3. 拖橙色十字：改变 IK 目标。
  4. 灰色手臂表示另一个可能解，说明 IK 可能有多解。
