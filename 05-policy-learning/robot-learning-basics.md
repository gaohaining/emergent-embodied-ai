# Robot Learning Basics

## 主要范式

- 模仿学习：从专家示范中学习策略。
- 强化学习：通过奖励和交互优化策略。
- 离线强化学习：从固定数据集中学习，减少真实交互成本。
- 行为克隆：直接拟合观察到动作的映射。
- 视觉语言动作模型：把视觉、语言和动作统一建模。
- 自监督学习：从无标注或弱标注机器人数据中学习表征。

## 关键变量

- 数据来源：人工遥操作、机器人自采集、仿真、视频、合成数据。
- 动作空间：关节、末端执行器、轨迹、离散动作 token。
- 观察空间：RGB、深度、点云、触觉、力矩、本体状态。
- 任务形式：单技能、长程任务、多任务、多机器人本体。

## 常见瓶颈

- 高质量真实机器人数据昂贵。
- 不同机器人本体之间迁移困难。
- 评测任务容易被过拟合。
- 长程任务中的错误会逐步累积。
