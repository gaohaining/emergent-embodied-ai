# Capability Ladder

## L0: Remote Operation

机器人主要由人类远程控制，智能系统用于感知辅助、稳定控制或数据记录。

## L1: Single Skill Automation

在固定环境中完成单一技能，例如抓取、开门、导航、分拣。

## L2: Multi-Skill Policy

能在有限任务集合内选择和组合技能，对环境变化有一定鲁棒性。

## L3: Language-Conditioned Task Execution

能根据自然语言目标，在已知场景中完成多步任务。

## L4: Open-World Generalization

面对新物体、新布局、新任务表达和有限干扰时，仍能稳定完成任务。

## L5: Continual Embodied Learning

能在真实部署中持续学习，主动采集数据，发现失败并改进策略。

## 当前要跟踪的断点

- 单任务成功率到多任务泛化。
- 短程技能到长程任务。
- Demo 环境到真实部署。
- 单一本体到跨本体迁移。
- 人工数据采集到自主数据飞轮。
