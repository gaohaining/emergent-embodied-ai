# 08 Hardware And Deployment

硬件与部署解决的问题是：如何让智能系统真实、安全、稳定地运行在机器人上。

## 初学者先读

1. [hardware.md](hardware.md)：机器人平台、传感器和执行器。
2. [deployment.md](deployment.md)：真实部署、安全、延迟和运维。

## 硬件影响什么

- 能做什么动作。
- 能承受多大负载。
- 感知是否足够可靠。
- 数据采集成本。
- 策略是否能跨平台迁移。
- 系统是否能安全部署。

## 部署关注什么

- 实时性和延迟。
- 安全边界和急停。
- 人工接管。
- 日志和回放。
- 模型版本管理。
- 失败检测和恢复。

## 初学者要记住

具身智能的能力不是只由模型决定。硬件、传感器、控制、安全和运维会共同决定系统能不能真正落地。
