# 传感器

## 它解决什么问题

传感器决定机器人能知道什么。模型再强，如果感知不到接触、位置或状态，也很难稳定完成任务。

## 传感器类型

| 传感器 | 提供什么 | 常见用途 |
| --- | --- | --- |
| RGB 相机 | 颜色、纹理、语义 | 物体识别、场景理解、VLA 输入 |
| 深度相机 | 距离和粗几何 | 抓取、避障、空间定位 |
| 双目相机 | 视觉和深度估计 | 机器人操作、移动感知 |
| LiDAR | 大范围几何 | 导航、建图、避障 |
| 触觉传感器 | 接触形状、压力 | 灵巧操作、滑动检测 |
| 力/力矩传感器 | 接触力和扭矩 | 装配、插入、力控 |
| 本体传感器 | 关节角、速度、电流 | 控制闭环、状态估计 |
| 麦克风 | 声音和语音 | 人机交互、环境事件 |

## 设计取舍

- 传感器越多，系统越复杂。
- 视觉语义强，但接触状态弱。
- 触觉对操作关键，但数据难规模化。
- 多传感器融合需要同步、标定和延迟管理。

## 进阶笔记

传感器选择会影响数据格式，也会影响模型可迁移性。一个依赖特定触觉传感器的策略，可能很难直接迁移到没有触觉的机器人。

## 开放问题

- 通用机器人最小传感器栈是什么？
- 触觉是否会成为灵巧操作突破的关键？
- 多模态大模型如何可靠融合实时传感器数据？
